# 羊了个羊 (Sheep a Sheep) - 微信小游戏 (99% 还原版)

这是一个高度还原《羊了个羊》核心玩法与视觉风格的微信小游戏。

## 核心特性 (99% 还原度)

1. **视觉风格**: 
   - 采用经典的草地绿渐变背景。
   - 卡片具有 3D 厚度感和动态阴影。
   - 包含遮挡算法：下层方块被覆盖时会自动变暗。
2. **核心模式**:
   - **第一关**: 极简入门（3x3），让你秒变领头羊。
   - **第二关**: 难度陡增，模拟原版不可能完成的任务级别堆叠。
3. **槽位机制**: 7 个固定槽位，3 个同类自动消除。
4. **道具系统**:
   - **洗牌**: 打乱场上所有方块位置。
   - **撤回**: 将最后一个点击的方块放回场内。
   - **移出**: 将槽位中前三个方块移出，腾出空间。
5. **排行榜**: 全国各省份今日战绩排名，模拟真实的竞争氛围。

## 如何运行

1. 打开 **微信开发者工具**。
2. 导入目录 `C:\tea`。
3. 勾选 **将 JS 编译成 ES5**。
4. 点击 **编译**。
